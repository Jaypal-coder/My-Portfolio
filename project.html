Stone-Paper-Scissor Game

This is a simple and interactive Rock Paper Scissor game developed using HTML, CSS, and JavaScript. The project provides an engaging user interface where players can select one of the three hand signs — Rock, Paper, or Scissor — by clicking on visually appealing icons.

Upon clicking the "Play Your Move" button, the game randomly generates the computer's move and displays the result. It also keeps track of both the player’s and computer’s scores, updating them after each round.
